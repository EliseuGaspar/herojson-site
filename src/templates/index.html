O Pacote HeroJson é um pacote leve. A sua dependência é de um modúlo padrão do python(Json), o que significa que não há porque não tê-lo!

Além de ser facíl de se instalar, é também facíl de se usar... Não é necessário um curso para usar este pacote. O que é preciso é apenas saber ```ler uma documentação```.

O que é o HeroJson ?

O HeroJson é um modúlo python para manipulação de arquivos json.
Este modúlo depende do modúlo json do python, ou seja não é necessário muita coisa para usa-lo.

HeroJson /

Como Usa-lo?

Aqui em baixo está um exemplo simples de como o pacote HeroJson funciona!

Crie um arquivo example.json no diretório raíz(pode ser um outro qualquer)

{

"nome" : "Eliseu",

"idade" : 17

}

Depois do nosso example.json ter sido criado, podemos criar agora o nosso app.py

from herojson import JsonHero

#

json = JsonHero( path = "./example.json", encoding = "UTF-8" )

json.add( key = "altura", value = 1.87 ) # Adiciona uma nova chave ao arquivo json.

Agora o nosso arquivo example.json deverá estar assim:

{

"nome" : "Eliseu",

"idade" : 17,

"altura" : 1.87

}

Vamos mais uma vez mexer em nosso app.py

from herojson import JsonHero

print(json.key( key = "nome" )) # Retorna o valor da chave passada se esta existir.

>>> " Eliseu "

Agora modificando mais uma vez o nosso example.json

from herojson import JsonHero

json.drop( key = "idade" )

Agora o nosso arquivo example.json deverá estar assim:

{

"nome" : "Eliseu",

"altura" : 1.87

}

E este foi um simples exemplo de como o herojson pode ajudar a manipular um arquivo json.
