Conhecendo melhor o HeroJson

Ao ser chamado a instância do HeroJson necessita de dois paramatros: o path e o encoding. Veja o exemplo abaixo!

from herojson import JsonHero

json = JsonHero( path = "./example.json", encoding = "UTF-8" )

Descrevendo os seus metódos:

key() : Este metódo retorna o Valor da chave passada por parametro se esta existir, e None se não.

{

"nome" : "Eliseu",

"idade" : 17

}

from herojson import JsonHero

print(json.key( key = "nome" )) # Retorna o valor da chave passada se esta existir.

>>> " Eliseu "

from herojson import JsonHero

print(json.key( key = "peso" )) # Retorna o valor da chave passada se esta existir.

>>> " None "

add() : Este metódo adiciona um novo par de chave e valor se estes não existirem ainda!

from herojson import JsonHero

json = JsonHero( path = "./example.json", encoding = "UTF-8" )

json.add( key = "peso", value = "55kg" )

{

"nome" : "Eliseu",

"idade" : 17,

"peso" : "55kg"

}

drop() : Este metódo deleta um par de chave e valor se existirem.

from herojson import JsonHero

json = JsonHero( path = "./example.json", encoding = "UTF-8" )

json.drop( key = "peso" )

{

"nome" : "Eliseu",

"idade" : 17

}

update() : Este metódo atualiza o valor de uma chave se esta existir.

from herojson import JsonHero

json = JsonHero( path = "./example.json", encoding = "UTF-8" )

json.update( key = "nome", key = "Gaspar" )

{

"nome" : "Gaspar",

"idade" : 17

}

AllKeys() : Este metódo retorna uma lista com os nomes de todas as chaves contidas no arquivo json.

from herojson import JsonHero

json = JsonHero( path = "./example.json", encoding = "UTF-8" )

print(json.AllKeys())

>>> ["nome","idade"]

AllValues() : Este metódo retorna uma lista com os nomes de todos os valores contidos no arquivo json.

from herojson import JsonHero

json = JsonHero( path = "./example.json", encoding = "UTF-8" )

print(json.AllValues())

>>> ["Gaspar",17]

Content() : Retorna o conteúdo do arquivo em forma de string.

from herojson import JsonHero

json = JsonHero( path = "./example.json", encoding = "UTF-8" )

print(json.Content())

>>> {"nome":"Gaspar","idade":17}
